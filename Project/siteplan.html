Website Planning Document

IoT-Based Energy Monitoring System for SMEs

Site Name

SmartEnergy Monitor

This name reflects a modern and intelligent system designed to monitor energy usage. It combines "Smart" (technology-driven) and "Energy Monitor" (core purpose of the system).

Site Purpose

The website provides information about an IoT-based energy monitoring system that helps Small and Medium Enterprises (SMEs) track electricity usage in real time. It also offers a dashboard simulation, system explanation, and user interaction through forms.

Scenarios

  • How can I monitor my electricity usage in real time?
  • Which devices in my business consume the most power?
  • How can I reduce my electricity costs using this system?

Color Scheme

  • Dark Blue (#0f172a) – Background color
  • Green (#22c55e) – Buttons, highlights, headings
  • Gray (#1e293b) – Cards and sections

Typography

  • Segoe UI – Used for body text
  • Arial – Backup font
